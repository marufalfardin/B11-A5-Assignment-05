Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM, such as, getElementById() for ID, getElementsByClassName() for class, getElementsByTagName() for tag, querySelector() for CSS-style selector, and querySelectorAll() to select multiple elements matching a selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets HTML content, innerText sets visible text, and textContent sets all text including hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is when we add one event listener to a parent element to handle events for its child element.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and then move up to its parent element, triggering any event listeners on those parent
